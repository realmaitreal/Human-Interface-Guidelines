# Selection and input

- [Color wells](Selection%20and%20input/Color%20wells.md) — A color well lets people adjust the color of text, shapes, guides, and other onscreen elements.
- [Combo boxes](Selection%20and%20input/Combo%20boxes.md) — A combo box combines a text field with a pull-down button in a single control.
- [Digit entry views](Selection%20and%20input/Digit%20entry%20views.md) — A digit entry view fills the entire screen and prompts people to enter a series of digits, like a PIN, using a digit-specific keyboard.
- [Image wells](Selection%20and%20input/Image%20wells.md) — An image well is an editable version of an image view.
- [Pickers](Selection%20and%20input/Pickers.md) — A picker displays one or more scrollable lists of distinct values that people can choose from.
- [Segmented controls](Selection%20and%20input/Segmented%20controls.md) — A segmented control is a linear set of two or more segments, each of which functions as a button.
- [Sliders](Selection%20and%20input/Sliders.md) — A slider is a horizontal track with a control, called a thumb, that people can adjust between a minimum and maximum value.
- [Steppers](Selection%20and%20input/Steppers.md) — A stepper is a two-segment control that people use to increase or decrease an incremental value.
- [Text fields](Selection%20and%20input/Text%20fields.md) — A text field is a rectangular area in which people enter or edit small, specific pieces of text.
- [Toggles](Selection%20and%20input/Toggles.md) — A toggle lets people choose between a pair of opposing states, like on and off, using a different appearance to indicate each state.
- [Virtual keyboards](Selection%20and%20input/Virtual%20keyboards.md) — On devices without physical keyboards, the system offers various types of virtual keyboards people can use to enter data.
