# Navigation and search

- [Path controls](Navigation%20and%20search/Path%20controls.md) — A path control shows the file system path of a selected file or folder.
- [Search fields](Navigation%20and%20search/Search%20fields.md) — A search field lets people search a collection of content for specific terms they enter.
- [Sidebars](Navigation%20and%20search/Sidebars.md) — A sidebar appears on the leading side of a view and lets people navigate between areas of your app or top-level collections of content, like folders and playlists.
- [Tab bars](Navigation%20and%20search/Tab%20bars.md) — A tab bar lets people navigate between top-level sections of your app.
- [Token fields](Navigation%20and%20search/Token%20fields.md) — A token field is a type of text field that can convert text into *tokens* that are easy to select and manipulate.
