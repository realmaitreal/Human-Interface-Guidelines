# Content

- [Charts](Content/Charts.md) — Organize data in a chart to communicate information with clarity and visual appeal.
- [Image views](Content/Image%20views.md) — An image view displays a single image — or in some cases, an animated sequence of images — on a transparent or opaque background.
- [Text views](Content/Text%20views.md) — A text view displays multiline, styled text content, which can optionally be editable.
- [Web views](Content/Web%20views.md) — A web view loads and displays rich web content, such as embedded HTML and websites, directly within your app.
